SQL CHEAT SHEET

SELECT

Used to select the desired data

SELECT column_name, column_name
FROM table_name;

To return the whole table...

SELECT * FROM table_name;

WHERE

Returns the corresponding data based on a given value

SELECT column_name, column_name
FROM table_name
WHERE column_name operator value;

ORDER BY

Returns values with the ability to sort columns in ascending or descending order.

SELECT column_name, column_name
FROM table_name
ORDER BY column_name ASC|DESC, column_name ASC|DESC;

LIKE

Used in conjunction with Where to essentially do the same thing.

SELECT column_name(s)
FROM table_name
WHERE column_name LIKE pattern;

UPDATE

Updates the table

UPDATE table_name
SET column1=value1,column2=value2,...
WHERE some_column=some_value;

DELETE

Deletes rows of the table

DELETE FROM table_name
WHERE some_column=some_value;

CREATE TABLE

Create a table by specifying column_name, data_type, and size parameter.

CREATE TABLE table_name
(
column_name1 data_type(size),
column_name2 data_type(size),
column_name3 data_type(size),
....
);

DISTINCT

Distinct returns values that only have one instance.

SELECT DISTINCT column_name,column_name
FROM table_name;

INNER JOIN

Inner Join will return every row from both tables but only if the compared columns are the same. You'll most likely use this when dealing with primary and foreign keys.

SELECT column_name(s)
FROM table1
INNER JOIN table2
ON table1.column_name=table2.column_name;

Join, in the example below, is the same exact thing as Inner Join.

SELECT column_name(s)
FROM table1
JOIN table2
ON table1.column_name=table2.column_name;

#4- Enumerable: The #cycle Method

February 22, 2016

The Enumerable Module

The Enumerable module provides a set of methods useful to Ruby's collections classes, such as Array and Hash. As a module, Enumerable allows for the use of its methods by any class that has an each method, which proves the class' position as a collection class. The Enumerable module's boundary-ignorant accessibility, therefore, gives users a handy set of tools that work for any Ruby collection object.

The #cycle Method

cycle is a popular method of the Enumerable module. It iterates a given block of code over each element any specified number of times. If not given a number, it will loop infinitely. For example, 

a = ['a','b','c']

          a.cycle(2) { |letter| print letter }

          =>abcabc

          a.cycle { |letter| print letter }

          =>abcabcabcabcabcabcabcabc...
After iterating over each element of the collection, the method will reset and start from the beginning, hence the name, "cycle."

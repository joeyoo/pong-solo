What is a Class?

A class is a template of attributes and behaviors that provides the framework for which an object is created. Objects of the same class have the same attributes and can also use the same methods defined within the class. Most likely, you'll have learned about methods in Ruby before classes. And, since everything in Ruby is an object, a good way to think of a class is as a physical blueprint. Within this blueprint is the basic layout of characteristics and functions that you can use to create any instance of this blueprint. In Ruby, the characteristics can be the attributes and the functions can be the methods. For example, let's make a class called Car. We'll give Car the basic attributes of type and color. For it's functions/methods, we'll give it accelerate and brake.

To define the new class, we wrote.

class Car
The first thing we do is initialize the object with the parameters we decided on: (type, color). Because we initialize parameters, a new Car object will need those arguments. In this method, you see @type and @color. The @ symbol denotes an instance variable. An instance variable is a variable that can be accessed in every instance of an object. In this case, if we wanted to, we can actually access our instance variables from our methods. You may have noticed the 
attr_accessor :type
attr_accessor :color
This method is the syntactic sugary combination of what are called reader and writer methods. These replace the need to define a method to read or write a whole method. So, if you wanted to only be able to read the @type, you would write:
attr_reader :type
If you wanted to only edit the instance variable, you would write:
attr_writer :type
The colon(:) next to the variable name makes it a symbol, which links it to the instance variable we have in the initialization.

Now we create an Car object and call it Hummer. Notice how we can use .type and .color to access the type and color of the car. Since we used attr_accessor, we can also write. If we want to change the color to pink, we can do this:

me:> Hummer.color('pink')
to get
=> 'pink'
